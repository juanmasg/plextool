# plextool

```
usage: plextool.py [-h] --plex HOST:PORT [--ipy IPY] [--list-shows] [--diff-scrape-tmdb] [--title TITLE]

optional arguments:
  -h, --help            show this help message and exit
  --plex HOST:PORT, -P HOST:PORT
                        HOST:PORT
  --ipy IPY             Run ipython at the end
  --list-shows, -S      List all plex shows
  --diff-scrape-tmdb    Check missinng episodes by parsing the mdb website
  --title TITLE, -t TITLE
                        Filter by title
```
